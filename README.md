# 安装说明

1、右键点击 x86 / x64 / ARM 安装包，点击属性，切换到"数字签名"选项卡。双击签名列表中的"esaeten"。在弹出的"数字签名详细信息"窗口中点击"查看证书"，再点击"安装证书"。

2、在"证书导入向导"中：

- "存储位置"选"本地计算机"，下一步。（需要同意UAC权限请求）

- 选择"将所有的证书都放入下列存储"，点击"浏览"，选择"受信任人"，确定，下一步，继续完成证书导入。

3、双击 x86 / x64 / ARM 安装包安装。
